Governo Britânico Libera o repositório Puppet do site GOV.UK

January 21, 2016

O governo britânico liberou todo o seu código Puppet responsável por instalar, configurar e manter as aplicações do site GOV.UK. Você pode acessar esse post para conferir a notícia, o código encontra-se no github e no puppet-forge.

Antes de liberar o código, foi realizado um intenso trabalho para remover todas as credenciais que pudessem existir nos manifests de módulos puppet, essas credenciais foram colocadas em local externo, protegendo assim as aplicações.

A iniciativa aconteceu pois existe um norma de governo chamada “Point 8” que está presente no “Digital by Default Service Standard” do governo Britânico. Essa norma estabelece que todo o código dos serviços de governo deve ser ‘aberto’ e ‘reutilizável’. Além disto, existe também outro documento chamado “Desing Principle” que em seu décimo princípio recomenda manter as soluções abertas pois isto as faz melhores - “Make things open: it make things better”.

Segundo o blog gdstechnology, a partir de agora o desenvolvimento será todo realizado e centralizado no github.
