Powerline shell no osx

July 1, 2014

Eu já falei da powerline aqui no blog, mas naquela ocasião era para uso interno no vim, recentemente passei a utilizar a powerline no bash através do projeto powerline shell.

A instalação é rápida e fácil, logue no console e digite

$ cd ~
$ git clone https://github.com/milkbikis/powerline-shell

Agora vamos instalar

$ ./install.py

Pronto, agora vamos criar um link simbólico.

$ ln -s ~/powerline-shell/powerline-shell.py ~/powerline-shell.py

Feito isto vamos configurar nosso arquivo .bash_profile

$ vim ~/.bash_profile

Adicione estas linhas ao início do arquivo e comente qualquer configuração da variável PS1 que existir no arquivo.

function _update_ps1() {
	export PS1="$(~/powerline-shell.py $? 2> /dev/null)"
	}

export PROMPT_COMMAND="_update_ps1; $PROMPT_COMMAND"

Recarregue o arquivo

$ source .bash_profile

Pronto, sua powerline estará funcionando.
